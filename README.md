# 📝 JavaScript Review for React

Repaso de JavaScript moderno como preparación para React. Cubre desde fundamentos del lenguaje hasta manipulación del DOM, organizado en 32 archivos `.js` progresivos y dos páginas HTML de práctica.

## 🗂️ Estructura del proyecto

```
javascript-review-for-react/
├── css/
│   └── output.css
├── js/
│   ├── 01.js - 32.js
│   └── funciones.js
├── DOM.html
└── index.html
```

## 📋 Contenido por archivo

| Archivos | Tema |
|---|---|
| `01 - 02` | Variables: `let` y `const` |
| `03` | Tipos de datos: `string`, `number`, `boolean`, `null`, `undefined`, `BigInt`, `Symbol` |
| `04 - 07` | Objetos: destructuring, spread operator, `Object.assign`, object literal enhancement |
| `08` | Template strings y concatenación |
| `09 - 11` | Arrays: creación, destructuring |
| `12` | Iteración de arrays: `forEach`, `map`, `for...of` |
| `13 - 16` | Funciones: declaration, expression, arrow functions y retorno de valores |
| `17` | Métodos de arrays: `filter`, `includes`, `some`, `find`, `every`, `reduce` |
| `18 - 21` | Condicionales: `if/else`, comparadores estrictos, operadores lógicos y ternarios |
| `22` | Optional chaining (`?.`) y nullish coalescing (`??`) |
| `23` | Evaluación de cortocircuito (`&&`) |
| `24` | Módulos ES: `import` / `export` (named y default) |
| `25 - 27` | Fetch API: Promises, async/await, `Promise.all` y comparación de rendimiento |
| `28 - 32` | Manipulación del DOM: selectores, eventos (click, input, submit) |

## 🚀 Cómo ejecutar el proyecto

No requiere instalación ni dependencias adicionales.

1. Clonar o descargar el repositorio.
2. Abrir `index.html` o `DOM.html` directamente en el navegador.
3. Para probar un archivo .js específico, cambiar el src del <script> en el HTML correspondiente.
